How things got to where they are now...

I have been interested in creating a personal blog since I was in my second year at University, back in 2017. I gave a try at making it on wordpress once, but it didn't feel personal enough. I realized that I didn't just want to have my own blog, I wanted to create it personally, like i engineered it with a little lazy computer programming skills. So I collected all the documents and images and waited for the opportunity to do this.

I was able to get a great opportunity to re-attempt this in 2020 when I saw the blog of a young  undergraduate from the 17 batch; Ramith Hettiarachchi, who also happens to be a SEF ScholarX member of 2019.

Ramith had a very attractive blog, which looked very personal and well developed. I would like to extend my personal thanks to Ramith for his advice and support, in helping me write about this experience on "HOW I MADE MY PERSONAL SELF-HOSTED BLOG USING GHOST AND GATSBY".

PLEASE NOTE: I did everything on my Laptop which was running Windows 10. I didn't dual boot and get linux thanks to the amazing done Windows Linux Subsystem (WSL). Also, I did everything for FREE!

What I wanted to make here?

I was very excited about the year 2020, even though (so far) it was one of the most terrible years in Human History. This was solely due to 2020 opening new avenues for me, in various fields such as AI, Blogging and Product Development Research to name a few. I wanted to pick up some new skills I could use in my future career as well.

Ramith had used this API called Ghost, which was powerful and unique for publishing online. Together with gatsby, it gave the complete FRONT-END framework I required for my custom blog.

The entire blog had to be hosted on Github, and then I could build and deploy my new project with Netlify, and it will be hosted on the internet like any other Blog.

Basically, When I started to create this new blog, I found out that these websites here very influential for my work.

  1. https://www.conordewey.com/blog/building-my-blog-with-gatsby-and-ghost/
  2. https://josebrowne.com/tutorial-static-blog-using-headless-ghost-2-0-gatsby-netlify/
  3. https://www.devopscycle.com/ghost-gatsby-netlify-jamstack/
PLEASE NOTE: They used to do this kind of blog creation using a package called buster, but it is outdated and not available for Ubuntu 20.04 LTS. Our approach will go with Netlify instead of Buster! Much simpler!

Install Ubuntu 20.04 on Windows 10 using WLS

This was the big jewel that made this all possible. Thanks to the Windows Linux Subsystem (WLS), I was able to get this up and running like a charm.

This website https://itsfoss.com/install-bash-on-windows/ had the steps on how to install WLS on my computer. It was so simple!

PLEASE NOTE: Make sure to RESTART the computer when you install Ubuntu 20.04 LTS on Windows 10.

Now if you have Ubuntu running,  add your user name and your password and get into your linux terminal. 

sudo apt-get update && sudo apt-get upgrade

OPTIONAL: https://docs.microsoft.com/en-us/windows/wsl/wsl2-install

There is a small issue in WSL that you need to correct using the following code snippet.  

sudo add-apt-repository ppa:rafaeldtinoco/lp1871129
sudo apt update
sudo apt install libc6=2.31-0ubuntu8+lp1871129~1 libc6-dev=2.31-0ubuntu8+lp1871129~1 libc-dev-bin=2.31-0ubuntu8+lp1871129~1 -y --allow-downgrades
sudo apt-mark hold libc6

Install yarn

This is what I installed first. Simply followed the instructions given here. 

curl -sS https://dl.yarnpkg.com/debian/pubkey.gpg | sudo apt-key add -
echo "deb https://dl.yarnpkg.com/debian/ stable main" | sudo tee /etc/apt/sources.list.d/yarn.list
sudo apt update && sudo apt install yarn
yarn --version
